How to Export:

  1. Navigate to any page (Transactions, Accounts, Budgets)
  2. Apply filters if you want specific data
  3. Click the "Export" button
  4. Choose your format (CSV, JSON, Excel)
  5. File downloads automatically
API Export: Use /api/v1/export/<format>?type=all for programmatic access.
